# User Personas

## Persona 1: 고객 (Customer) - "김민수"

| 항목 | 내용 |
|------|------|
| 이름 | 김민수 |
| 나이 | 30대 |
| 역할 | 매장 방문 고객 |
| 기술 수준 | 일반 (스마트폰/태블릿 기본 사용 가능) |

**목표:**
- 대기 없이 빠르게 주문하고 싶다
- 메뉴를 편하게 탐색하고 선택하고 싶다
- 주문 상태를 실시간으로 확인하고 싶다

**동기:**
- 직원 호출 없이 자유롭게 주문
- 주문 실수를 줄이고 싶음 (장바구니에서 확인 후 주문)

**불편 사항:**
- 복잡한 UI, 느린 반응 속도
- 주문 후 상태를 알 수 없는 상황

## Persona 2: 매장 관리자 (Admin) - "박서연"

| 항목 | 내용 |
|------|------|
| 이름 | 박서연 |
| 나이 | 40대 |
| 역할 | 매장 운영자/사장 |
| 기술 수준 | 중급 (PC/태블릿 업무 사용 가능) |

**목표:**
- 들어오는 주문을 실시간으로 파악하고 싶다
- 테이블별 주문 현황을 한눈에 보고 싶다
- 메뉴를 쉽게 관리하고 싶다

**동기:**
- 인건비 절감 및 운영 효율성 향상
- 주문 누락 방지
- 매장 회전율 향상 (빠른 테이블 정리)

**불편 사항:**
- 주문 누락이나 지연
- 복잡한 관리 도구
